# Documentacao do Projeto

Esta pasta concentra a documentacao operacional e tecnica do repositorio.

## Arquivos principais

- `DOCUMENTACAO_COMPLETA.html`: guia consolidado do projeto, com visao funcional, tecnica e de publicacao.
- `architecture/README.md`: decisao arquitetural e contrato de independencia entre Quiz e Trilha.
- `CHECKLIST-QA.md`: roteiro manual de verificacao antes de publicar ou apresentar.
- `media/`: imagens e midias usadas na documentacao.

## Quando usar cada documento

- Leia `DOCUMENTACAO_COMPLETA.html` para onboarding completo.
- Consulte `architecture/README.md` antes de reorganizar arquivos ou alterar deploy.
- Rode o `CHECKLIST-QA.md` quando houver mudanca visual, de acessibilidade ou de fluxo.
